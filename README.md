# Daily Code Challenges

A collection of my daily coding practice from various platforms to improve problem-solving skills and maintain consistent coding habits.

## Overview

This repository serves as a centralized location for tracking progress across multiple coding challenge platforms.

## Platforms

### [freeCodeCamp](./freecodecamp)
Interactive coding lessons and certifications covering web development, data science, and more.

### [LeetCode](./leetcode)
Algorithm and data structure problems for technical interview preparation.

## Goals

- Practice consistently
- Improve problem-solving skills
- Build a portfolio of solved challenges
- Track progress over time

## Structure

Each platform has its own directory with:
- Individual challenge solutions
- Platform-specific README with progress tracking
- Organized by topic, difficulty, or curriculum path

## Getting Started

Navigate to any platform directory to view challenges and solutions for that platform.
